# Arbuz Bot

Discord-бот для ежедневного розыгрыша роли **АРБУЗ** на сервере.

## Что умеет бот

- Пользователь кидает `!arbuz`
- Обычный бросок: от **1 до 20**
- Если выпадает **1**, пользователь может выбрать:
  - `!да` — включить арбузную анархию
  - `!нет` — оставить результат `1`
- Арбузная анархия тоже кидает от **1 до 20**
- Если в арбузной анархии выпадает **20**, роль **АРБУЗ** за этот день **не получает никто**
- Роль определяется **только по обычным броскам**
- Если по обычным броскам ничья, запускается **тай-брейк**
- Тай-брейк:
  - не попадает в общий топ
  - не считается обычным броском нового дня
  - проходит без анархии
  - длится до **12:00 МСК следующего дня**

## Логика ролей

### Обычный день
1. Пользователь использует `!arbuz`
2. Бот кидает число от `1` до `20`
3. Если выпало не `1`, результат сразу считается обычным броском дня
4. Если выпало `1`, бот ждёт `!да` или `!нет`

### Анархия
- `!да` запускает отдельный анархический бросок `1..20`
- Этот результат **не участвует** в борьбе за роль
- Если в анархии выпало `20`, роль за этот день **не выдаётся никому**

### Тай-брейк
Если в конце дня несколько пользователей выбили одинаковый максимум среди **обычных** бросков:
- запускается тай-брейк
- в тай-брейке участвуют только эти пользователи
- тай-брейк не считается обычным броском нового дня
- тай-брейк не попадает в `!top`
- дедлайн тай-брейка — **12:00 МСК следующего дня**

## Команды

### `!arbuz`
Основная команда броска.

### `!да`
Включить арбузную анархию после выпадения `1`.

### `!нет`
Отказаться от анархии и оставить результат `1`.

### `!top`
Показывает:
- топ обычных бросков
- результаты анархии отдельно
- распределение бросков
- активный тай-брейк, если он есть

## Технические детали

### Часовой пояс
Бот использует часовой пояс:

- `Europe/Moscow`

### База данных
Используется SQLite.

Файл базы данных:

- `bot.db`

### Таблицы
- `rolls` — дневные броски пользователей
- `tiebreaks` — тай-брейк раунды
- `meta` — состояние тай-брейка и служебные данные
